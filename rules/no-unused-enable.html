# disallow unused eslint-enable comments (eslint-comments/no-unused-enable)

  • 🌟 The "extends": "plugin:eslint-comments/recommended" property in a configuration file enables this rule.

This rule warns eslint-enable directive-comments which have no effect.

# Rule Details

Examples of 👎 incorrect code for this rule:

/*eslint eslint-comments/no-unused-enable: error */ /*eslint-disable no-undef */ doSomething() /*eslint-enable no-undef-init */
Now loading...
/*eslint eslint-comments/no-unused-enable: error */ doSomething() /*eslint-enable */
Now loading...

Examples of 👍 correct code for this rule:

/*eslint eslint-comments/no-unused-enable: error */ /*eslint-disable no-undef */ doSomething() /*eslint-enable no-undef */
Now loading...
/*eslint eslint-comments/no-unused-enable: error */ doSomething()
Now loading...
