# disallow eslint-disable comments without rule names (eslint-comments/no-unlimited-disable)

  • 🌟 The "extends": "plugin:eslint-comments/recommended" property in a configuration file enables this rule.

eslint-disable directive-comments disable all rules by default. This may cause to overlook some ESLint warnings unintentionally. So you should specify the rules to disable accurately.

# Rule Details

Examples of 👎 incorrect code for this rule:

/*eslint eslint-comments/no-unlimited-disable: error */ var foo; //eslint-disable-line
